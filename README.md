# Playwright Automation Testing

## Description
This project demonstrates automated UI testing using Playwright with TypeScript.
It follows the Page Object Model (POM) to create maintainable and scalable tests.

## Tools Used
- Playwright
- TypeScript
- Node.js

## How to Run Tests
1. Install dependencies
   npm install

2. Run tests
   npx playwright test

## Test Coverage
- Login (success and failure)
- Add item to cart
- Add multiple items
- Remove item (inventory and cart page)
- Verify item appears in cart
